# electron-vite

An Electron application with Vue and TypeScript

## Recommended IDE Setup

采用pinia实现状态管理，使用axios发送http请求

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
